# CoverDownloaderInline для PulseSync

Аддон для скачивания обложки трека

## Возможности

* Автоматическое добавление кнопки скачивания рядом с названием трека
* Кнопка отображается при наведении курсора
* Если установлен аддон CopyTrackName — кнопка располагается правее иконки копирования
* Скачивание обложек треков одним кликом в максимальном качестве (оригинал)
* Настройка качества изображений
* Настройка формата имени файлов
* Настройка размера и прозрачности иконки
* Уведомления о скачивании

## Настройки

### Основные настройки

**Качество изображения**
* Низкое (200x200) - для быстрого скачивания
* Среднее (400x400) - оптимальный баланс
* Высокое (1000x1000) - отличное качество
* Оригинал (максимум) - оригинальный размер без сжатия
* По умолчанию: высокое

### Сохранение файлов

**Формат имени файла**
* Настройте шаблон имени файла
* Доступные переменные:
  * `{artist}` - имя исполнителя
  * `{title}` - название трека
* По умолчанию: `{artist} - {title}`

### Внешний вид

**Размер иконки**
* Размер иконки скачивания в пикселях (12–24)
* По умолчанию: 18

**Прозрачность иконки**
* Прозрачность иконки в неактивном состоянии (5–100%)
* По умолчанию: 70%

### Дополнительно

**Показывать уведомление**
* Отображает уведомление при успешном скачивании
* По умолчанию: включено
